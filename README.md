##初期化
####Mac OS X

    cd mac
    ./initialize

####Windows

PowerShellで

    cd windows
    ./initialize.ps1

##更新
####Mac OS X

    cd mac
    ./update

####Windows

PowerShellで

    cd windows
    ./update.ps1

####Neovim
プラグインはlazy.nvimで自動管理されます。
手動更新が必要な場合はNeovimで

    :Lazy update
